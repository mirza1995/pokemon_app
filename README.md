# Pokemon application in React, Typescript using Vite

## Getting Started

### Required versions

##### npm version: 8+
##### node.js version: 14.18+

### Install

Install dependencies.

```bash
npm install
```

Serve with hot reload at <http://localhost:5173>.

```bash
npm run dev
```

### Lint

```bash
npm run lint
```

### Typecheck

```bash
npm run typecheck
```

### Build

```bash
npm run build
```

### Test

```bash
npm run test
```
